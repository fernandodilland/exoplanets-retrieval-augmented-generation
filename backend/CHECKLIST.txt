FASE 1: LIMPIEZA
────────────────

[ ] 1. Presiona Win + R
[ ] 2. Escribe: appwiz.cpl

[ ] 4. Busca "Python 3.13"

FASE 2: DESCARGA
────────────────

[ ] 7. Abre: https://www.python.org/downloads/
[ ] 8. Descarga: Python 3.12.7 (64-bit)
        ⚠️ NO descargues Python 3.13


FASE 3: INSTALACIÓN
───────────────────

[ ] 9. Ejecuta el instalador descargado
[ ] 10. ✅ MARCA: "Add Python to PATH"  ← ¡IMPORTANTE!
[ ] 11. ✅ MARCA: "Install for all users" (opcional)
[ ] 12. Clic en: "Install Now"
[ ] 13. Espera a que termine (2-3 minutos)
[ ] 14. Clic en: "Close"


FASE 4: VERIFICACIÓN
────────────────────

[ ] 15. Cierra TODAS las ventanas de VS Code
[ ] 16. Abre VS Code de nuevo
[ ] 17. Abre una nueva terminal (Ctrl + Shift + `)
[ ] 18. Ejecuta: python --version
        ✅ Debe mostrar: Python 3.12.7


FASE 5: CONFIGURACIÓN DEL PROYECTO
──────────────────────────────────

[ ] 19. En la terminal, ejecuta:
        cd C:\git\exoplanets-retrieval-augmented-generation\backend

[ ] 20. Ejecuta el script de configuración:
        setup-environment.bat

        Este script hace:
        ✓ Verifica Python
        ✓ Crea el entorno virtual
        ✓ Instala pip actualizado
        ✓ Instala todas las dependencias


FASE 6: EJECUCIÓN
─────────────────

[ ] 21. Ejecuta el servidor:
        run-dev.bat

[ ] 22. Verifica que se inicie correctamente
        ✅ Debe mostrar: "Application startup complete"
        ✅ Puerto: http://0.0.0.0:2090

[ ] 23. Abre en el navegador:
        http://localhost:2090/docs
        ✅ Debe mostrar la documentación de la API
